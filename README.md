# ⚽ Football Player Name Game

A fun multiplayer game where you compete against AI to name football players by letter!

## 🎮 How to Play

1. AI randomly selects a letter (A-Z)
2. You name a professional footballer starting with that letter
3. AI responds with another player
4. No repeating players allowed!
5. Say "I AM OUT" if you can't name a player
6. Last one standing WINS!

## 📊 Features

- **Real player database** - Uses your CSV dataset
- **AI opponent** - Computer plays against you
- **No repeats** - Tracks used players
- **Score tracking** - See who's winning
- **Beautiful UI** - Modern football-themed design

## 🚀 How to Use Your Own Dataset

1. Place your CSV file in the same folder
2. Name it `players.csv`
3. Make sure it has a column with player names
4. Open `index.html` in your browser
